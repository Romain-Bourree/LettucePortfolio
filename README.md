# Lettuce Portfolio

Personal portfolio — single-page site with an interactive work timeline and motion-driven UI.

React, TypeScript, and Vite. Animation with Framer Motion and GSAP; audio via Howler / use-sound.

```bash
npm install && npm run dev
```

Production build: `npm run build` (output in `dist/`).
